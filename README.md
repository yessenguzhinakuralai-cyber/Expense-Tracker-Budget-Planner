**FinMate bot: Personal Finance Telegram Bot**

**FinMateBot** is a robust Telegram-based financial assistent designed to help users track their daily finances, manage monthly budgets, and perform complex loan calculations. Built with Python, it focuses on user-centric design and data-driven financial insights. 

**##KEY FEATURES:**:
***Dynamic Expense Tracking:*** Users log expenses by selecting a category (Entertainment, Cafe, Shopping, etc.) and then entering the amount.
***Precise Logging:*** Every transaction is automatically timestamped with the exact date and time for audit-ready history.
***Comprehensive Statistics:*** Generates daily summeries and all-time statistics, including total income, expenses, and net balance.
***Mortgage Calculator:*** A professional tool that calculates monthly annuity payments, total interest, and total payout.
***Intelligent Budget Planning:*** Allows users to set a monthly spending limit with real-time alerts if they exceed it.
***Financial Advice:*** Provides context-aware financial tips, including real-world banking facts (e.g., Freedom Bank deposits) and wealth management rules like the 50/30/20 principle. 

**##Technical Stack:**
***Language:*** Python 3.x
***Library:*** pyTelegramBotAPI (Telebot)
***Data Storage:*** JSON-based persistent storage for user profiles and transaction history
